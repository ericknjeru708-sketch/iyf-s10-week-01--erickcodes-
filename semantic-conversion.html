Task 2.1: Semantic HTML Conversion

Bad (Non-Semantic) Version

<div class="header">
  <div class="logo">My Site</div>
  <div class="nav">...</div>
</div>
<div class="main">...</div>
<div class="footer">...</div>

Good (Semantic) Version

<header>
  <h1>My Site</h1>
  <nav>
    <ul>
      <li><a href="#">Home</a></li>
      <li><a href="#">About</a></li>
      <li><a href="#">Contact</a></li>
    </ul>
  </nav>
</header>

<main>
  <article>
    <h2>Welcome to My Site</h2>
    <p>This is my first paragraph...</p>
  </article>
  <aside>
    <h3>Recent Posts</h3>
  </aside>
</main>

<footer>
  <p>&copy; 2026 Erick Njeru</p>
</footer>

Key Changes: Replaced <div> tags with proper semantic tags: <header>, <nav>, <main>, <article>, <aside>, <footer>.
